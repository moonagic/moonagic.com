##### 编译依赖
```bash
apt-get install libcurl4-gnutls-dev libexpat1-dev gettext zlib1g-dev libssl-dev
```
##### 下载
到[Github](https://github.com/git/git/releases)下载需要的版本
##### 安装
```bash
autoconf
./configure prefix=/usr/local all
make
make install
```
